# Online-Food-Delivery-Website

## Project Description
This is a frontend-based Online Food Delivery Website created as a team project.  
The website allows users to browse food items, view offers, and place orders after logging in.

## Features
- Responsive homepage design
- Animated login system
- User authentication using LocalStorage
- Food menu with pricing
- Deals & offers section
- Online order system
- Location access
- Contact form
- Logout functionality
- Mobile-friendly navigation

## Technologies Used
- HTML5
- CSS3
- JavaScript
